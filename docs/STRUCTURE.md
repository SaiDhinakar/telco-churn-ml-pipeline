# Code Structure

```
telco-churn-mlops-pipeline/

├── configs 		-> configurations
├── data		-> datasets raw, processed (cleaned)
│   ├── processed
│   └── raw
├── deployment		-> contains the model inference code
│   ├── api		-> API endpoint with versions v1
│   │   └── v1
│   ├── models		-> pydantic models for data validations
│   ├── scripts		-> automation scripts for setup and start the server
│   └── services	-> ML services
├── docs		-> Documentations
├── logs		-> Inference logs
├── mlartifacts		-> ML atrifacts generated by mlflow
├── mlruns		-> ML run generated by mlflow
├── notebooks		-> Model development and mlflow tracking using notebooks
├── tests		-> Test scripts
└── utils		-> Contains utils for application

```
